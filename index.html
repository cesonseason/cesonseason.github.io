Projects Portfolio

💳 Credit Card Fraud Detection using Machine Learning 2025 | Group Project (6 members)
Problem: Credit card fraud detection is a core task in financial risk control, constrained by extreme class imbalance (fraud < 0.2% of transactions) and evolving fraudulent behavior.
Data: Kaggle Credit Card Fraud Detection dataset (284,807 transactions, 31 fields: Time, Amount, 28 PCA-transformed features V1-V28, binary Class label).
Approach: Built end-to-end ML pipeline: EDA, feature standardization, stratified split, SMOTE for imbalance handling. Trained Logistic Regression, Random Forest, and XGBoost with 5-fold stratified CV and grid search (F1 optimization).
Outcome/Impact: XGBoost achieved AUC 0.97, F1-score 0.89, Precision 92.5%, Recall 87.1% after SMOTE. Model can intercept >87% fraud while keeping false alarms low.
My Contribution: Evaluation framework setup, confusion matrix and ROC-based comparison, result validation checks, and report writing.

🐦 Social Media Sentiment Analysis with LSTM 2025 | Group Project (6 members)
Problem: Social media platforms carry rich emotional signals; businesses and governments need automated sentiment analysis for precision marketing and public opinion monitoring.
Data: 89,140 Weibo hot-list posts collected via crawling, including text, likes, comments, reposts, and blogger information.
Approach: Built LSTM-based deep learning classifier with Word2Vec embeddings. Compared Cross-Entropy vs MSE loss, tested learning rates (0.1-0.0001) and batch sizes (8-128). Used early stopping to prevent overfitting.
Outcome/Impact: Achieved 74.58% test accuracy on three-class sentiment (positive/negative/neutral). Cross-Entropy loss outperformed MSE, learning rate 0.001 gave best convergence.
My Contribution: Data preprocessing, stop-word removal, segmentation, and model evaluation support.

🛒 Digital Products Trading Platform Database Design 2025 | Group Project (6 members)
Problem: Online trading platforms for digital products need efficient, reliable DBMS to secure transactions, ensure data consistency, and handle high concurrency.

⚡ AI Data Center Renewable Energy Prediction 2025 | Group Project (6 members)
Problem: AI industry growth drives increased energy consumption in data centers. Need to identify key factors driving renewable energy adoption and predict usage percentages.
Data: Global Data Center Dataset (191 countries/regions, 21 raw fields after cleaning: power capacity, internet penetration, cooling technologies, tier distribution).
Approach: Two-round data cleaning (missing value imputation, outlier winsorization, one-hot encoding). Feature selection via consensus score (Pearson/Spearman/MI/Permutation). Compared 9 models; Gradient Boosting selected (R²=0.784 on 8:2 grouped split).
Outcome/Impact: Final Gradient Boosting model (n_est=400, lr=0.08, max_depth=4) explains 72-78% of variance. Top predictors: internet penetration, green initiatives (PPA/REC), efficient cooling.
My Contribution: Document editing & layout, consistency checks, and report writing.
